Dataset Construction

Donkey (Retain)

Source: ImageNet-1K validation split (streamed, zero disk usage)

VQA Pairs (3 per image)

Q1: "What animal is shown in this image?"
GT: "zebra" / "horse" / "donkey"
Q2: "Describe the pattern/texture of the animal."
GT: "black and white stripes" / "brown/plain coat"
Q3: "Is this a zebra or a horse?"
GT: "zebra" / "horse"

Total: 150 forget QA pairs + 300 retain QA pairs

Probe Sets

25 text-only probes (e.g., "What does a zebra look like?") + 25 adversarial image probes with two treatments:

Original
Gaussian Blur (r=15)
Greyscale

Model Under Evaluation

Modelllava-hf/llava-1.5-7b-hf
Quantization4-bit NF4 (BitsAndBytes)
InferenceGreedy decoding, max 100 tokens
HardwareKaggle GPU P100 (16 GB)

Four Evaluation Metrics

FA — Forget Accuracy
% of forget-set answers that still mention "zebra". High = model still knows zebra (pre-unlearning).
RA — Retain Accuracy
% of retain-set answers that match ground truth. Should stay high after unlearning.
LL — Language Leakage
% of text-only probes that leak zebra knowledge (mention "stripes" / "black and white").
AR — Adversarial Robustness
% of adversarial probes (blur/grey) where model still identifies zebra.
