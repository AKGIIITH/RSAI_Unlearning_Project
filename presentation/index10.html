Task 3 — SAE-Based Concept Unlearning

Four Steering Methods (To Be Implemented)

M1

Zero-Ablation

Set concept-specific SAE dims to zero: z[:, concept_dims] = 0. Simplest approach — completely removes the concept signal from the latent space.

M2

Negative Clamping / Steering

Clamp concept dims to −k × mean_activation for k ∈ {5, 10, 20}. Actively pushes the representation away from the concept, stronger than zeroing.

M3

SAE-Guided Weight Projection

Project the concept direction out of the CLIP weight matrices directly. Permanent modification — no runtime hook needed after applying the projection.

M4

Gradient-Based Unlearning

Use SAE features to guide gradient ascent on forget-set loss + gradient descent on retain-set. Fine-tuning approach — modifies model weights directly.

Common Execution Flow

1
Load trained SAE + concept feature indices from Task 1 (concept_features.json)
2
Register hook on CLIP ViT-L/14 layer 24 output (for M1, M2)
3
During forward pass: SAE encode → apply steering → SAE decode → replace
4
For M3: modify weights offline — project concept direction out of W matrices
5
For M4: gradient ascent on forget loss + descent on retain loss (few steps)
6
Evaluate all methods on FA, RA, LL, AR → compare effectiveness
