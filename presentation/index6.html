Methodology

Three-Stage Pipeline Overview

Task 1 — SAE & Concept Features

Train SAE + Extract Concept Features

1
Extract CLIP ViT-L/14 vision features from 10K ImageNet images (layer 24, 1024-dim)
2
Train Sparse Autoencoder (1024 → 4096 → 1024) with L1 sparsity on encoder activations
3
Pass zebra images → SAE → record mean activation per latent dim
4
Compute differential activation (zebra − horse) → top-K = concept features
Done — Results suboptimal
Task 2 — Baseline & Datasets

Build Baseline + Forget/Retain Sets

1
Stream ImageNet-1K → 80 zebra (forget) + 50 horse + 50 donkey (retain)
2
Create VQA pairs per image (3 questions each): identification, pattern, classification
3
Build adversarial probes (blur + greyscale on 25 zebra images)
4
Evaluate LLaVA-1.5-7B (4-bit) on all sets → FA, RA, LL, AR metrics
Complete
Task 3 — Concept Unlearning

SAE-Based Feature Steering at Inference

1
Load concept features (top-K zebra-specific SAE dims) from Task 1
2
Hook into CLIP vision encoder → intercept layer-24 activations
3
Apply steering: zero-ablation, negative clamping (−k×mean), weight projection, or gradient-based
4
Re-evaluate FA, RA, LL, AR → compare methods & measure unlearning effectiveness
